TNC Animal Detector

This application runs object detection algorithm on images to detect the presence
of animals in the images.
Please begin the application by clicking the start button below.

Start

Instructions

Step 1: Start

Start the application by clicking on the Start button in the top right corner.

Step 2: Upload

Upload the images that you want detected. Wait for the images to get uploaded and click begin.

Step 3: Wait

Wait a few minutes for the program to finish. More the images, more the wait!

Step 4: Downlaod

Once the program is complete, you can download the labeled images and a supplementary CSV file. The download link is only valid for one-time use.
